# The Power of Tech

This is a public news site for anyone to go and view and understand the significance of technology as explained by its background, what are the caveats and disadvantages, and what is next for technological advancement? 

There is also a feature for users to compare any stock (preferably tech stock) to the S&P 500 to see how much the tech sector has grown compared to the rest of the market, and how much unprecedented value it has created.

## Website Link

https://xmire814.github.io/csafinalproject/index.html

## Features

Understand technology through:
1. Background which covers the impact on quality of life, and global wealth explained by the stock market.
3. Caveats section that explains the disadvantages such a great technology possesses.
4. What is the next section to explain why it is important that tech constantly improves and what the next technological advancements are predicted to be.
5. Finally a stock chart to compare a stock to the S&P 500 by generating a graph. This is to understand the significant difference in growth, wealth, and power of the tech sector compared to the rest of the market. 

## Installation

In order for the website to work, there must be a free API Key generated from: https://www.alphavantage.co/support/#api-key 

To run this project locally:
1. Clone the repo
2. Navigate to project folder via terminal
3. Run: python3 -m http.server 8000
4. Open: http://localhost:8000

Through Github: 

1. Clone repo and open it
2. Go to Settings > Pages.
3. Under "Build and deployment", select:
   - Source: Deploy from a branch
   - Branch: main

GitHub will publish the site at:

https://yourusername.github.io/repository-name/

This project can be deployed through any platform by following its instructions.
